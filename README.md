# Kembar Futagroup

Website company profile Kembar Futagroup berbasis Nuxt 4 dan Nuxt UI.

## Setup

Install dependencies:

```bash
npm install
```

Jalankan development server:

```bash
npm run dev
```

Build production:

```bash
npm run build
```

Validasi kode:

```bash
npm run lint
npm run typecheck
```
